## 🛡️ 四层风控架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    四层风控引擎 (绝对否决权)                       │
│                                                                 │
│  ┌── L1 静态限额 (<1ms) ────────────────────────────────────┐  │
│  │ • 白名单校验    • 单笔名义限额 (10~100,000 USDT)         │  │
│  │ • 价格偏离检查  • 可交易性校验                            │  │
│  └─────────────────────────────────────────────────────────┘  │
│                              │                                  │
│                              ▼                                  │
│  ┌── L2 实时敞口 (<2ms) ────────────────────────────────────┐  │
│  │ • 同标的最大敞口    • 下单频率限制 (10秒/10单)            │  │
│  │ • 杠杆上限 (加密20x/美股4x)  • 单仓上限 10%              │  │
│  │ • 总敞口上限 50%                                         │  │
│  └─────────────────────────────────────────────────────────┘  │
│                              │                                  │
│                              ▼                                  │
│  ┌── L3 后台回撤 (1s轮询) ──────────────────────────────────┐  │
│  │ • 最大回撤 15% → FLATTEN    • 日内亏损 5% → halt_all()  │  │
│  │ • 保证金率 80% → REDUCE                                    │  │
│  └─────────────────────────────────────────────────────────┘  │
│                              │                                  │
│                              ▼                                  │
│  ┌── L4 熔断器 (事件驱动) ──────────────────────────────────┐  │
│  │ CLOSED →(连续5次失败)→ OPEN →(60秒)→ HALF_OPEN →(成功)→ CLOSED │
│  │ 熔断中: 拒绝所有订单                                       │  │
│  └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│  四态决策: APPROVE(批准) / REJECT(拒绝) / REDUCE(减仓) /        │
│           FLATTEN(强平)                                         │
│  审计: 纳秒戳 + 状态快照 + 只增不改                              │
└─────────────────────────────────────────────────────────────────┘
```
