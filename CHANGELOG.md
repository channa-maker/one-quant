## [0.1.0] — 2026-06-30

### 项目初始发布 — 阶段一~六完整实现

本版本完成 ONE量化 系统从地基到上线的全阶段架构搭建与核心代码实现。

---

### 阶段一 · 地基

#### 工程脚手架与 CI/CD
- 初始化项目目录结构，Python 3.12 + hatchling 构建体系
- Ruff + mypy strict 代码质量配置
- Docker Compose 开发环境编排（PostgreSQL / TimescaleDB / Redis / ClickHouse / MinIO）
- GitHub Actions CI 流水线（lint → 类型检查 → 测试 → 覆盖率）
- pydantic-settings 配置管理（`ONE_` 前缀环境变量）
- 结构化日志（JSON + 中文摘要 + `log_mask` 脱敏）

#### EventBus 与消息契约
- EventBus 抽象 + Redis Pub/Sub 实现 + InMemoryEventBus（测试用）
- 统一消息信封 `{channel, schema_version, seq, ts_ns, trace_id, data}`
- 关键通道 Redis Streams 持久化（订单/成交/风控不丢可重放）

#### 行情网关（加密）
- 领域类型定义（Ticker / Kline / OrderBook / Trade / OptionQuote）
- 币安 WebSocket 行情接入（tick + L2 增量 + K 线）+ 归一化
- OKX WebSocket 行情接入 + 归一化
- 断线重连（指数退避 ≤ 60s）+ 快照对齐 + 缺口回补
- 行情多源冗余 + 自动切换

#### 全量数据采集与数据湖（Medallion 架构）
- `one-collector` 进程 + Bronze 层原始落盘（Parquet / ClickHouse）
- 数据质检门（乱序丢弃 + 跳变标记 + 缺口检测 + 去重）
- Silver 层清洗对齐（事件时间对齐、归一化）→ TimescaleDB + ClickHouse
- Gold 层因子计算骨架 + 特征商店（Parquet 离线 + Redis 在线）
- 数据血缘记录（因子 → Silver → Bronze → 源）
- tick 级历史回放接口
- 冷热分层 + ZSTD 压缩 + 对象存储归档策略
- 已退市/下架标的全历史保留 + Point-in-Time 数据快照
- 标的主数据 Instrument Master + 跨所符号映射
- 新标的上线校验流程（流动性/数据/风控/合规四项）

#### 接口契约定型
- `Strategy` ABC + `Signal` 类型（策略接口契约）
- `RiskRule` 协议 + `ExchangeAdapter` / `UnifiedBroker` ABC
- `Factor` / `ExecutionAlgo` / `DataSource` / `Agent` / `ScreenerModel` 协议 + 注册表
- `Notifier` 协议 + 前端 `Panel` 接口约定
- `@register_*` 装饰器注册机制

#### 监控骨架
- Prometheus 指标埋点（行情新鲜度 / 队列深度 / 延迟分段）
- Grafana 中文大盘（系统健康 + 数据质量）
- OpenTelemetry 链路追踪骨架

### 阶段二 · 交易闭环

#### 策略引擎与回测框架
- 策略注册表 + runner 主循环（单 asyncio 事件循环）
- 回测引擎（tick 级，事件时间，成本/滑点模型）
- 回测一致性校验（回测/实盘偏差 < 0.05%，无未来函数检查）
- 基础策略 ×3：EMA 交叉 / RSI 反转 / 网格
- 两层出场（固定出场 + ExitBrain 影子）+ 重启状态恢复

#### 四层风控引擎
- L1 静态限额（白名单 / 名义 / 可交易性）
- L2 实时敞口（最大敞口 / 频率 / 价格偏离 / 杠杆上限，硬编码常量）
- L3 后台回撤 kill switch（1s 轮询）+ 全局日内止损总闸 `halt_all()`
- L4 熔断器（关闭 / 打开 / 半开三态）
- 四态决策 + 不可变审计（纳秒戳 / 状态快照，只增不改）
- 混沌工程测试（订单洪流 / 价格跳空 / 交易所断连）

#### 执行引擎（OMS/EMS）
- OMS 订单状态机 + 幂等（`clientOrderId` UUIDv4）
- 币安 / OKX 交易适配器（下单 / 撤单 / 持仓 / 对账）
- 客户端限流 `rate_limiter.py`
- EMS 算法拆单（TWAP / VWAP / POV / 冰山）+ 滑点控制
- 持仓恢复（重启拉真实持仓 + 审计重建）
- 模拟盘（同代码同行情，仅撮合模拟）
- 多策略净额轧差（内部对冲）+ 跨策略同向限额

#### 账户/资金会计
- 复式记账 Ledger + 实时盯市（已实现/未实现盈亏分离）
- 多币种 NAV 合并 + FX 折算统一基准币种
- 每日对账闭环（账本 vs 交易所对账单，差异告警）

### 阶段三 · AI 引擎 + 自进化

#### LLM 智能体
- 多 Provider 接入（Claude / DeepSeek + local 预留）+ token 计量 + 日预算硬上限
- 智能体体系：简报官 / 哨兵 / 分诊员 / 解读员 / 情绪 / 宏观 / 多空辩论组
- RAG 向量库（新闻 / 研报 / 历史决策）+ 机构记忆

#### ML 选股选币
- 因子库（动量 / 资金流 / 链上 / 波动 / 情绪 / 事件）+ 因子命名规范
- ML 打分模型（XGBoost / LightGBM 排序）+ 训练管线 + MLflow 注册
- LLM 复核层 + 候选池输出（分数 / 理由 / 置信度）+ SHAP 可解释

#### 自进化平台
- 策略全生命周期闭环（10 环节）编排
- 冠军-挑战者机制（影子 PK + 自动晋升）
- 自动再训练（滚动 + 概念漂移检测）+ 模型版本灰度/回滚
- AI 数据 → 分析 → 对比 → 进化闭环 + 中文进化报告

#### AI 信号推荐与评分系统
- 证据源子信号抽取框架（`SignalSource` 协议 + 注册表）

- 信号卡数据模型 + 历史同类检索（向量）
- 信号生命周期对账 + 权重/校准回流

#### 模型治理与 AI 防护
- 模型风险管理 MRM（模型清单 / 独立验证 / 模型卡 / 审批 / 退役）
- AI 数据投毒防护（新闻源可信度加权 + 多源交叉验证）
